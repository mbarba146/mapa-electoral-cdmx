# Mapa Electoral CDMX

Aplicación web analítica para explorar patrones históricos de votación en la Ciudad de México por alcaldía.

## Objetivo

Construir una aplicación web que permita visualizar, comparar y descargar información electoral histórica de la CDMX, usando datos simulados en la primera versión y dejando preparada la arquitectura para integrar datos oficiales posteriormente.

## Pregunta principal

¿La Ciudad de México vota de forma homogénea o existen zonas con comportamientos políticos claramente distintos?

## Alcance inicial

- Dashboard electoral por alcaldía.
- Comparación histórica entre años.
- Comparador de alcaldías y partidos.
- Datos simulados para prototipo.
- Página de noticias políticas simuladas por año.
- Documentación del proceso de aprendizaje.
- Estructura preparada para React, FastAPI, Docker y PostgreSQL.

## Stack objetivo

- Frontend: React + Vite + TypeScript
- Backend: FastAPI
- Datos iniciales: CSV / JSON
- Base futura: PostgreSQL
- Contenedores: Docker + Docker Compose
- Gestión: GitHub Issues + GitHub Projects

## Estado del proyecto

Fase inicial: definición, estructura del repositorio y documentación base.
