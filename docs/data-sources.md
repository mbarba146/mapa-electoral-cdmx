# Data Sources

## Versión inicial

La primera versión usará datos simulados.

## Fuentes planeadas para versión real

- Resultados electorales oficiales del INE / IECM.
- Indicadores sociodemográficos de INEGI.
- Noticias abiertas mediante GDELT u otra fuente pública.

## Nota

Los datos del prototipo no representan resultados oficiales.
