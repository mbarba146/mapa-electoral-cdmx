# Technical Decisions

## Decisión 1: usar datos mock primero

### Contexto

El proyecto tiene una fecha de entrega corta.

### Decisión

Usar datos simulados en la primera versión para validar flujo, estructura y experiencia de usuario.

### Motivo

Reduce riesgo y permite avanzar en la aplicación antes de integrar datos oficiales.

## Decisión 2: separar noticias del análisis electoral

### Contexto

Las noticias pueden dar contexto, pero no deben interpretarse como causa directa del voto.

### Decisión

Crear una página independiente de noticias políticas por año.

### Motivo

Evita conclusiones incorrectas y mantiene neutralidad analítica.
